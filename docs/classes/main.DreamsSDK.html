Class DreamsSDK

DreamSDK is an utility class responsible for setting up and listening to messages being exchanged between the your context and Dreams iframe.

const sdk = new DreamsSDK('https://dreams.api.endpoint');
sdk.setup(callbacks);
sdk.start(jwk_token, locale);
